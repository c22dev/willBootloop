willBootloop?

The probability your phone will bootloop based on user reports.






"A bootloop is when an Apple device no longer boot after a reboot, instead remaining on the Apple logo permanently. Usually these are unfixable, and updating/restoring to a new iOS version is the only solution."

- whitetailani


Probabilities are calculated off app's latest release download count. Misleading scores can be achivied if release is brand new. If Github API limits your fetching ability, app will fallback on bootloop count.


Please note that the developers may have fixed the bugs causing bootloops through updates.
